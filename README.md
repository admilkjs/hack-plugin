# Hack-Plugin

## 需要 Node.js v24+

## 功能

1. 为 `ICQQ` 添加发送 `type` 为 `raw` 的消息的能力
2. 支持发送假文件，格式 `#文件 name size`

## 后续会添加的功能

1. 取消息

## 安装方式

```bash
git clone https://github.com/admilkjs/hack-plugin ./plugins/hack-plugin
```

## 安装依赖

```bash
pnpm i
```

## 说明

本插件使用原生模块实现核心功能，已预编译 Linux 和 Windows 版本。
